# Calculadora
* Iremos utilizar o Visual Studio Code, programando em Java versão 16. 
* Precisaremos saber se a forma geométrica será 2D ou 3D.
* Informando os dados pertinentes para 2D:
  1. Área;
  2. Perímetro;
  3. Número de lados.
* Informando os dados pertinentes para 3D:
  1. Área; 
  2. Perímetro;
  3. Volume;
  4. Número de faces.
